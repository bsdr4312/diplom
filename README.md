# Backend-приложение для социальной сети (обмен фотографиями)


## Описание проекта

Серверная часть социальной сети, позволяющая пользователям:
- загружать публикации с фотографиями;
- комментировать публикации;
- ставить лайки публикациям.

Проект реализован на Django с использованием СУБД PostgreSQL.

## Требования к окружению

Перед установкой убедитесь, что у вас установлены:
- Python 3.9+;
- PostgreSQL 12+;
- Git;
- pip (менеджер пакетов Python).
